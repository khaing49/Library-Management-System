# 📚 Library Management System

This project is a back-end implementation of a simple Library Management System built using the **Spring Boot** framework. It provides a RESTful API to manage books, library members, and borrowing transactions.

## ✨ Features

* **CRUD Operations for Books:** Add, retrieve, update, and delete books.
* **CRUD Operations for Members:** Add, retrieve, update, and delete library members.
* **Transaction Management:** Record borrowing and returning books.
* **Availability Check:** Determine if a book is available for borrowing or when the next copy is due.
* **Data Persistence Ready:** Structured to use **Spring Data JPA** for database interaction.

## 🚀 Technologies Used

* **Language:** Java (JDK 24+)
* **Framework:** Spring Boot 3.x
* **Data Access:** Spring Data JPA
* **Build Tool:** Maven 
* **API Testing:** Postman (Recommended)
